Operación de Conjuntos: A - B

Conjunto A = {a, 2, c, 3, e, 4, g, 5, i, 6, k, 7, m}

Conjunto B = {a, b, c, d, e, f, g, h, i}

Procedimiento:

  1. Para calcular A - B, revisa cada elemento en A.
  2. Comprueba si el elemento existe en B.
  3. Si un elemento de A no se encuentra en B, agrégalo al conjunto resultante.
  4. El conjunto resultante será la respuesta de A - B
